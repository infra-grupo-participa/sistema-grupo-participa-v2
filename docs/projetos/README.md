# Documentos de produto

Diretorio canonico para PRDs, specs e documentos funcionais organizados por
projeto/modulo.

## Estrutura

```text
docs/projetos/
├── README.md
├── placas/
│   └── prd.md
└── depoimentos/
    ├── prd.md
    └── spec.md
```

## Regra de uso

- cada modulo ganha sua propria pasta
- `prd.md` e o documento de produto principal
- `spec.md` guarda detalhamento tecnico quando existir
- evitar duplicatas do mesmo documento em caminhos paralelos
